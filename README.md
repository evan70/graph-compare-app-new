# Grafy 2024

Moderná webová aplikácia pre vizualizáciu dát pomocou Chart.js a ApexCharts.

## Funkcie
- Interaktívne grafy
- Podpora viacerých typov grafov
- Responzívny dizajn
- Svetlý/tmavý režim
- Fullscreen mód
- Mobilná optimalizácia
- Vlastné farebné palety
- Moderná navigácia

## Aktuálny stav
- Verzia: 0.1.0 (2024-01-23)
- Status: Vo vývoji
